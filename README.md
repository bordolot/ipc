# IPC 

Pure rust, minimal, zero-dependecy, local Unix IPC library.

## Features
- signals (in progress)
- System V:
    - shared memory(shmget/mmap) (in progress)
    - message queues(msgget) (in progress)
    - semaphores (in progress)
- unix domain sockets (UDS) (in progress)
- memory mapped files (mmap) (in progress)
- eventfd (in progress)
- signalfd (in progress)
- timerfd (in progress)
